El comando uname -a muestra información detallada sobre el sistema operativo, incluyendo el nombre del sistema, la versión del kernel, la arquitectura y otros datos técnicos.

El comando whoami muestra el nombre del usuario que tiene iniciada la sesión actualmente.

Esta información es útil para administrar servidores porque permite identificar el sistema operativo utilizado, verificar qué usuario está ejecutando acciones y facilitar tareas de configuración, mantenimiento y seguridad.
